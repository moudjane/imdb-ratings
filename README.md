# IMDB Ratings Project

## Setup

### 1. Get OMDb API Key
Get a free API key from [omdbapi.com](https://www.omdbapi.com)

### 2. Create .env File
Create a `.env` file in the `backend` directory with your actual API key:
```
OMDB_API_KEY=your-api-key
```

## Running the Application

### Method 1: Docker (Recommended)
The simplest way to run the project without installing anything on your machine.

```bash
docker-compose up --build
```
